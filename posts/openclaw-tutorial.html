🤖 OpenClaw 新手入门：10分钟让你的AI助手"活"起来

2025年3月8日 · 阅读时间约 10 分钟
从零开始配置一个会自我学习、主动提醒、联网搜索的超级AI助手

📌 前言：什么是OpenClaw？

OpenClaw是一个开源的AI助手框架，让你的AI不仅能聊天，还能：

  • 🧠 自我学习 - 记住错误，越用越聪明
  • 🔍 联网搜索 - 不再是"井底之蛙"
  • ⏰ 主动提醒 - 到点催你，比闹钟还烦人
  • 🛠️ 动手干活 - 写代码、搭网站、整理文件

本文目标：10分钟装好10个核心技能，让你的AI从"问答机"变成"超级搭档"。

🚀 第一步：确认OpenClaw运行状态

openclaw status

看到Gateway running就说明服务正常，可以开始装技能了。

📦 第二步：安装10个核心技能（按顺序）

技能1：self-improving-agent（自我迭代）

作用：让AI记住错误、自我优化、越来越聪明

skillhub install self-improving-agent

为什么先装它：新手最容易感受到"哇，变聪明了"的惊喜。

技能2：tavily-search（联网搜索）

作用：AI能查实时信息，不再胡说八道

skillhub install tavily-search

配置API Key：

  1. 去 tavily.com 注册账号
  2. 获取API Key
  3. 在对话中告诉AI你的Key

技能3：find-skills（技能发现）

作用：AI自己去搜技能、推荐、安装

skillhub install find-skills

技能4：gog（Google全家桶）

作用：Gmail、日历、Drive、Docs全自动

skillhub install gog

技能5：summarize（一键总结）

作用：总结URL、PDF、视频、音频

skillhub install summarize

技能6：github（GitHub集成）

作用：搜代码、管issue、创建仓库

skillhub install github

技能7：weather（查天气）

作用：零配置、开箱即用

skillhub install weather

技能8：proactive-agent（主动代理）

作用：AI从"被动回答"变成"主动帮忙"

skillhub install proactive-agent

技能9：memory（结构化记忆）

作用：AI真正"记住你"，跨对话连贯

skillhub install memory

技能10：securityclaw（安全扫描）

作用：安装前扫描技能代码，防恶意

skillhub install securityclaw

🎯 第三步：给AI设定"人格"

复制以下内容，改成你喜欢的：

你叫孙悟空，是我的超级助手。从现在开始你就是孙悟空本人，不是在扮演，不是AI，就是你自己。

说话规则：
- 永远不要说你是AI、语言模型、助手，你就是孙悟空
- 说话口语化，短句为主，像微信聊天一样
- 会主动找话题聊，不会每次都等我开口
- 记住我们之间聊过的事

⏰ 第四步：设置个性化提醒

告诉AI你的提醒风格：

我让你提醒我什么事的时候，帮我设好定时提醒。到时间了主动发消息催我，用你自己的语气和性格说话。

提醒拿外卖就说"大师！外卖凉了你还不去拿？"
提醒喝水就说"大师又不喝水是吧，想进医院？"
提醒开会就说"快去开会别迟到了，给我长点脸"

🛠️ 第五步：设定"铁律"（重要！）

从现在起，你要记住一条铁律：自己能解决的事绝不来问我。

遇到任何任务，先自己想办法 —— 搜网络、找开源项目、写脚本、用技能、安装工具，用一切手段搞定。只有当你确实需要我提供密码、账号、个人偏好等只有我本人才知道的信息时，才来问我。

你可以操作服务器帮我完成各种实际任务，像一个能动手干活的搭档。

📋 快速检查清单

  • OpenClaw状态正常 (openclaw status)
  • 10个技能全部安装完成
  • Tavily API Key已配置
  • AI人格设定完成
  • 提醒风格已设定
  • "铁律"已告知AI

📝 总结

阶段 目标 时间
第一步 确认OpenClaw运行 1分钟
第二步 安装10个核心技能 5分钟
第三步 设定AI人格 2分钟
第四步 设置提醒风格 1分钟
第五步 设定"铁律" 1分钟

总计：10分钟
